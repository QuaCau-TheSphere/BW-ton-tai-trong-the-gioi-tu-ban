Khái niệm:: 
Các câu hỏi của bài nhân sự sẽ thay đổi nên cho phép nv chia sẻ mà không sợ bị lộ đề, còn của bên đào tạo thì không. Lý do ko đổi là để nhân viên nắm đúng được nội dung chính của khoá đào tạo. Ví dụ: khả năng thì nắm mất/ giảm thu nhập, phát sinh chi phí, thiện chí thì 3 cấp độ. Trước khi nộp bài mình có thể chụp lại các đáp án để giảng viên giải đáp.
