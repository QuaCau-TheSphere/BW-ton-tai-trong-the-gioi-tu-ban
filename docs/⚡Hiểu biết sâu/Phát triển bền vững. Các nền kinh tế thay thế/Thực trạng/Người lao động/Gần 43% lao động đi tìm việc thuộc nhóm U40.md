Người tìm việc nhiều nhất ở nhóm nghề:
- quản trị doanh nghiệp
- nhân viên bán hàng
- mua sắm và môi giới
- kinh doanh tiếp thị
- quảng cáo

Trong khi nhà tuyển dụng thường tìm lao động trong lĩnh vực:
- kỹ thuật
- vận tải
- bán hàng
