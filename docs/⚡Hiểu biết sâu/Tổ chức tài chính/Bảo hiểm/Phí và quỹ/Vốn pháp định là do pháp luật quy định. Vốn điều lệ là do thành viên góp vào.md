|              |                | Vốn pháp định | Vốn điều lệ |

| Nhân thọ     | Bao gồm + và   | 1000          | 1300        |
|              | Bao gồm + hoặc | 800           | 1000        |
|              | Trừ            | 600           | 750         |
| Phi nhân thọ | Bao gồm + và   | 300           | 500         |
|              | Bao gồm + hoặc | 250           | 450         |
|              | Trừ            | 200           | 400         |
| Sức khoẻ     |                | 300           | 400         |
