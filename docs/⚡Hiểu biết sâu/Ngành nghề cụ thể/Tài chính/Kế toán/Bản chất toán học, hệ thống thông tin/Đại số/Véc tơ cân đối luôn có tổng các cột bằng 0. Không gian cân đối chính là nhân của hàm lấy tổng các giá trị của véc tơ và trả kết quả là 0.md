Khái niệm:: 
Hàm $$σ: R^n → R$$ là hàm từ mô đun tự do $R^n$ vào miền được sắp $R$. $$σ(\textbf{v}) = \sum^n_{i=1}v_i$$

Nói cách khác, véc tơ cân đối là các véc tơ thỏa mãn $σ(\textbf{v}) =  0$. Nói cách khác, nó là $\text{Ker}(σ)$
