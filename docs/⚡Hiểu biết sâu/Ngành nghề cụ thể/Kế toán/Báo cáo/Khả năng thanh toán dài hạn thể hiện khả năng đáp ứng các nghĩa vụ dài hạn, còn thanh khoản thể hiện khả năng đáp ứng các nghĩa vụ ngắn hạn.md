Khái niệm:: 
Khả năng thanh toán dài hạn thể hiện khả năng đáp ứng các nghĩa vụ dài hạn của công ty, còn thanh khoản thể hiện khả năng đáp ứng các nghĩa vụ ngắn hạn. Tiền mặt được coi là phương tiện thanh toán có tính thanh khoản cao nhất. 

Một công ty thiếu thanh khoản có thể bị buộc phải phá sản dù có đầy đủ khả năng thanh toán dài hạn nếu nó không thể chuyển đổi tài sản của mình thành tiền để đáp ứng các nghĩa vụ tài chính.
